# PO-0017 — quotient algebra through order four

Frame: S05

Statement: At the paper's fixed orders three and four, exact quotient
extraction gives
`W3 = u1^3 W(v2,v3)` and
`W4 = u1^4 v2^3 w3^2 (w4/w3)'`; increasing translation columns reverse the
underlying arguments to `p4 < p3 < p2 < p1`.  The corresponding discrete
order-four determinant cascade has the same forward orientation.

Quantifiers and domains: real scalar row entries and quotient coordinates;
fixed sizes two, three, and four.

Hypotheses: none for the polynomial determinant identities.  The literal
quotient equations require only the named denominators to be nonzero.
Positivity is used only in separate sign corollaries.
