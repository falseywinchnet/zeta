# Atomic obligation records

`STATUS.md` is the initial 46-obligation ledger. Individual `PO-XXXX.md` files
are created when work on that obligation begins; each must use the schema in
`SPECIFICATIONS.MD` and may contain only one mathematical claim.

PO-0009/0010, PO-0017/0018/0020, PO-0022 through PO-0029, and PO-0030 through
PO-0041 are the first expanded records. The measure/CDF/transport core now has maintained Lean
theorems through PO-0041; PO-0009/0010, PO-0022 through PO-0031, and PO-0037
through PO-0041 are formally complete. PO-0032 through PO-0036 are retained as
`DEPRECATED_OPTIONAL` crossing records and excluded from the active
denominator. PO-0017/0018/0020 now record the
maintained fixed-order quotient/integral fragments and their unresolved actual-
kernel instance boundaries.
