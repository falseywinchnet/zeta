# Atomic obligation records

`STATUS.md` is the initial 46-obligation ledger. Individual `PO-XXXX.md` files
are created when work on that obligation begins; each must use the schema in
`SPECIFICATIONS.MD` and may contain only one mathematical claim.

PO-0030 through PO-0037 and PO-0018 are the first expanded records: the former
close the non-vacuous crossing proof conventionally, while the latter gives the
generic determinant-integral engine. Next records should split PO-0017 and
PO-0020 into their exact quotient and strictness claims.
