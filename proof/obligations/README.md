# Atomic obligation records

`STATUS.md` is the initial 46-obligation ledger. Individual `PO-XXXX.md` files
are created when work on that obligation begins; each must use the schema in
`SPECIFICATIONS.MD` and may contain only one mathematical claim.

PO-0030 through PO-0041 and PO-0018 are the first expanded records. The
measure/CDF/positive-transport core now has Lean fragments through PO-0041;
PO-0038/0039 isolate the still-open expectation and integration-by-parts object
identities. Next records should split PO-0017 and PO-0020 into their exact
quotient and strictness claims.
