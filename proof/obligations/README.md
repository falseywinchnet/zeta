# Atomic obligation records

`STATUS.md` is the initial 46-obligation ledger. Individual `PO-XXXX.md` files
are created when work on that obligation begins; each must use the schema in
`SPECIFICATIONS.MD` and may contain only one mathematical claim.

PO-0018, PO-0023 through PO-0029, and PO-0030 through PO-0041 are the first
expanded records. The measure/CDF/transport core now has maintained Lean
theorems through PO-0041; PO-0023/0024, PO-0026 through PO-0031, and PO-0037
through PO-0041 are formally complete. Next records should split PO-0017 and
PO-0020 into their exact quotient and strictness claims.
