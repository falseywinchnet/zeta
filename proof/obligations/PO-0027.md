# PO-0027 — exact coordinate sign bridge

Frame: S06

Statement:

`partial_xi Psi = -Q(p) N / Lambda^2`.

Formalization: `PF4.CoordinateSignBridge.coordinatePartialXiPsi_eq` multiplies
the actual left-coordinate derivative of `coordinatePsi` by the coordinate
speed `Q(p)` and proves equality with `PF4.Transport.partialXiExpression`.

Non-vacuity: the left side is a `deriv` of the independently defined function,
not a scalar assumed equal to the target expression. No sign is assumed.

Counterexample condition: an admissible ordered triple for which multiplying
the actual left-coordinate derivative by `Q(p)` has a different value or sign
orientation from `-Q(p)N/Lambda^2`.

Status: `FORMALLY_PROVED`.
