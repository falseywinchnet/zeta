# Advancement work

Each advancement round owns one immutable evidence directory:

```text
work/YYYY-MM-DD-short-round-name/
  ROUND.md
  notes.md
  src/
  data/
  results/
```

Only `ROUND.md` is mandatory. Record the question, model, date, starting MIND
factoids/tracemaps, methods, evidence, failures, conclusions, and open edges.
Everything said or produced during the round belongs here. Do not establish or
clean MIND factoids during the same round.

A later refine round may cite and atomize the work. The original directory stays
as the support record.
