# Preserved failed ansatz

The fifth generic collision coefficient of `J` is not generated by
`C4,C4',C4''` with the four natural scale-covariant jet monomials used in
`derive_collision_eps5.py`.  Exact polynomial coefficient comparison returns
an empty solution set.

Consequence: the planned statement “fourth and fifth collision remainders are
controlled only by `C4,C4',C4''`” is false.  A new sixth-jet invariant must be
retained and bounded.  This is a failure of a proposed compression, not a
negative PF4 result.
