# Preliminary draft archive

`strict_global_pf4_riemann_kernel.pdf` is the reviewed nine-page preliminary
draft created on 15 July 2026. Its SHA-256 checksum is

```text
a1cfb1437e92bbfe6bab3259e3b441429789d58aa4a1029087ce08885ecb8b94
```

`main.tex` is an archival transcription reconstructed from that PDF because
the generating TeX source was not present in the repository or alongside the
downloaded file. It preserves the theorem, section order, definitions, and
load-bearing displayed equations, but it is not asserted to reproduce line
breaks or PDF typography byte-for-byte.

This directory is immutable historical evidence. The corrected, maintained
paper is `../manuscript/main.tex`; review dispositions are in
`../PAPER_MAP.md`.

Known historical defect: PDF equation (30) omits the factor `Lambda` before
`T delta`. Equation (32) and the maintained proof use the correct numerator.
