\paragraph{Code and data availability.}

The public repository is
\url{https://github.com/falseywinchnet/zeta}. The exact analytic proof was
promoted in commit
\href{https://github.com/falseywinchnet/zeta/commit/8a4db9e24b3a47a3eb4d35b89eb1a3f8e80eab9d}{\texttt{8a4db9e24b3a}}.
That immutable snapshot contains the submission source, coefficient records,
environment lock, replay tool, and PDF; their SHA-256 values are listed in
\path{paper/RELEASE_MANIFEST.md}. The complete repository release is assigned
the DOI \url{https://doi.org/10.5281/zenodo.21404360}. The complete evidence replay is
\texttt{python scripts/replay\_paper.py} from the repository root.

All theorem-bearing sign decisions are exact. Numerical values produced by
the repository scripts are diagnostic only; there is no separate experimental
dataset.
